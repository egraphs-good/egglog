Struct TaggedRowBuffer

Source 
pub struct TaggedRowBuffer { /* private fields */ }
Expand description

A TaggedRowBuffer wraps a RowBuffer but also keeps track of a source RowId for the row it contains. This makes it useful for materializing the contents of a Subset of a table.
