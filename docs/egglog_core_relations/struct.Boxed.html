Struct Boxed

Source 
pub struct Boxed<T>(pub T);
Expand description

A newtype wrapper used to implement the BaseValue trait on types not defined in this crate.

This type is just a helper: users can also implement the BaseValue trait directly on their types if the type is defined in the crate in which the implementation is defined, or if they need custom logic for boxing or unboxing the type.
