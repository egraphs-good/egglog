Expand description

Read-side methods — name-indexed table lookup and iteration. Implemented for ReadState and FullState; not for PureState or WriteState (a Write context body must not depend on live DB state).

The single-entry methods (lookup, eclass_of, contains) return None if absent — never insert. The iteration / introspection methods (function_entries, constructor_enodes, table_size, table_sizes) walk the current contents of the database.

Detectable misuse (wrong table subtype, wrong arity) is reported as crate::ApiError via the method’s Result. Per-column sort matching is not checked at this layer.

fn lookup<K: IntoValues>( &self, name: &str, key: K, ) -> Result<Option<Value>, Error>

Look up a function’s output value at the given key. Returns Ok(None) if the row is absent. The returned Value is raw — extract a Rust type via Core::value_to_base for base sorts or Core::value_to_container for containers.

Only valid for function tables. Constructors error; use Read::eclass_of for those.

Source

fn eclass_of<K: IntoValues>( &self, name: &str, inputs: K, ) -> Result<Option<Value>, Error>

Look up a constructor’s eclass at the given inputs, without minting a fresh one on miss. Returns Ok(None) if absent.

Only valid for constructor tables. Functions error; use Read::lookup for those.

Source

fn contains<K: IntoValues>(&self, name: &str, key: K) -> Result<bool, Error>

True iff a row with the given key exists in the table. Works for any subtype — never mints.

Source

fn table_size(&self, name: &str) -> Option<usize>

Return the current row count for the named table, or None if no table with that name is registered.

Source

fn table_sizes(&self) -> Vec<(&str, usize)>

Snapshot the registered table names and their current row counts.

fn constructor_enodes( &self, name: &str, f: impl FnMut(Enode<'_>), ) -> Result<(), Error>

Call f on each Enode of a constructor / relation table. Errors with WrongSubtype if name is a function. To stop early, use Read::constructor_enodes_while.

Source

fn constructor_enodes_while( &self, name: &str, f: impl FnMut(Enode<'_>) -> bool, ) -> Result<(), Error>

Like Read::constructor_enodes, but stops as soon as f returns false.

fn function_entries( &self, name: &str, f: impl FnMut(FunctionEntry<'_>), ) -> Result<(), Error>

Call f on each FunctionEntry of a function table. Errors with WrongSubtype if name is a constructor. To stop early, use Read::function_entries_while.
