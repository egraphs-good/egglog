Expand description

Action-side write methods — name-indexed inserts/removes/subsumes plus union and panic. Implemented for WriteState and FullState; not for PureState or ReadState.

Detectable misuse (wrong table subtype, wrong arity) is reported as crate::ApiError via the method’s Result. Per-column sort matching is not checked at this layer.

Set a function table’s value at the given key — mirrors the egglog (set (f k) v) action.

Only valid for function tables. Constructors error; use Write::add for those.

fn add<R: IntoValues>(&mut self, name: &str, inputs: R) -> Result<Value, Error>

Mint or look up an eclass for a constructor — mirrors the egglog (Cons k1 k2 ...) expression form. Pass inputs only; the output eclass is minted (or returned if a row with these inputs already exists).

Only valid for constructor tables. Functions error; use Write::set for those.
